Souvenir Product Catalog in Virtual Reality

Dec 13, 2021
  • XR Types: Virtual Reality
  • Platform: Oculus Quest 1
  • Project Type: Private
  • Association: PT Arutala Digital Inovasi

  • Development Platform: Unity 2019.4 LTS
  • Framework: XR Interaction Toolkit version 0.9.4
  • Render Pipeline: Universal Render Pipeline

Hi everyone, VR Souvenir Catalog was my very first project when I work for PT Arutala Digital Inovasi. This is basically a virtual catalog store that allows consumer to move around and choose the products that they would like to buy. The player can grab the product, see the product information, and put / throw the product into the trolley. Unfortunately, the player still not be able to buy directly from the application.

This is my first time using XR Interaction Toolkit to develop VR application. Therefore, there are a lot of points that still can be improved on. This is also my first time using wheel collider to move the trolley around. I learned a lot while doing this project.
